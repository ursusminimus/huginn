# huginn

Set of simple tools for daily planning. Will be expanded. Open for new ideas.

Each tool is a standalone .html file with javascript. The goal is to have minimal dependencies and stay offline, let's see how that goes.

## Naming convention

Huginn *.html files are standalone htmls with javascript that you can open in any browser. Data is stored in the browser cache. The data can also be exported/imported into/from muninn *.csv or .json files as a backup or to have different variants.

## Available tools

- huginn tasks - simple todo app
- huginn tasks viz - burndown chart visualization of the huginn tasks dataset
- huginn cal - simple annual calendar
